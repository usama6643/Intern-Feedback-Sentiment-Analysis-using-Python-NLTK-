# Intern-Feedback-Sentiment-Analysis-using-Python-NLTK-
This project analyzes the emotional tone of internship feedback to improve the internship experience. The feedback data was collected from surveys, comments, and social media sources and analyzed using Python and the NLTK library.
The goal of this project is to evaluate intern opinions and classify their feedback into Positive, Neutral, and Negative sentiments.
Sentiment analysis helps organizations understand how interns feel about their experience and identify areas that require improvement.
In this project, intern feedback data is processed and analyzed using Python. The dataset contains multiple feedback comments from interns regarding their internship experience.

The NLTK VADER Sentiment Analyzer is used to calculate sentiment scores for each feedback entry. Based on these scores, the feedback is classified into three categories:

Positive

Neutral

Negative

The analysis helps highlight the overall satisfaction level of interns and provides insights for improving mentorship, training sessions, and workload management.
